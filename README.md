# ThaiHelpsThaiPlus
โปรแกรมคำนวณโครงการช่วยลดค่าครองชีพ 60/40 ไทยช่วยไทยพลัส

" กระทรวงการคลัง เปิดเผยรายละเอียดโครงการ 'ไทยช่วยไทยพลัส (60/40)'
สำหรับประชาชนทั่วไป 30 ล้านคน โดยกำหนดคุณสมบัติต้องมีสัญชาติไทย อายุ 18 ปีขึ้นไป
และไม่มีบัตรสวัสดิการแห่งรัฐ รัฐบาลจะสนับสนุนเงินร่วมจ่ายร้อยละ 60 (ไม่เกิน 200 บาท/วัน)
วงเงินคนละ 1,000 บาทต่อเดือน นาน 4 เดือน ย้ำวงเงินเหลือไม่ทบสิทธิข้ามเดือน "
