# ThaiHelpsThaiPlus (ไทยช่วยไทย พลัส)

โปรแกรมคำนวณโครงการช่วยลดค่าครองชีพ 60/40 "ไทยช่วยไทยพลัส" — Web Application รูปแบบ Landing Page เน้นใช้งานบนมือถือ (mobile-first) ทำงานล้วนด้วย HTML/CSS/JavaScript (ไม่มี build step, ไม่ต้อง login) และจดจำข้อมูลผู้ใช้ด้วย `localStorage`

🔗 ตัวอย่าง URL เมื่อ deploy: `https://<username>.github.io/ThaiHelpsThaiPlus/`

## รูปแบบโครงการ
"กระทรวงการคลัง เปิดเผยรายละเอียดโครงการ 'ไทยช่วยไทยพลัส (60/40)'
สำหรับประชาชนทั่วไป 30 ล้านคน โดยกำหนดคุณสมบัติต้องมีสัญชาติไทย อายุ 18 ปีขึ้นไป และไม่มีบัตรสวัสดิการแห่งรัฐ
- รัฐบาลจะสนับสนุนเงินร่วมจ่ายร้อยละ 60 (ไม่เกิน 200 บาท/วัน และไม่เกิน 1,000 บาท/เดือน)
- ประชาชนจ่ายเอง ร้อยละ 40

เช่น ซื้อสินค้า 300.- บาท รัฐบาลจะสนับสนุนเงิน 180.- เงินประชาชนจ่ายเอง 120.- (เงินรัฐบาลเหลือ 20.-/จาก 200 บาท --> วันนี้ และ เหลือ 820.-/จาก 1,000 บาท --> เดือนนี้)

วงเงินคนละ 1,000 บาทต่อเดือน นาน 4 เดือน ย้ำวงเงินเหลือไม่ทบสิทธิข้ามเดือน"

## คุณสมบัติ (Features)
- โลโก้ + หัวข้อ "ไทยช่วยไทย พลัส" + คำอธิบาย "ลดภาระ ลดค่าครองชีพ"
- แสดงวัน/วันที่/เดือน/ปี พ.ศ. แบบไทย พร้อมนาฬิกาเดินจริงรูปแบบ `hh:mm:ss` (24 ชั่วโมง)
- กล่องสถิติ "วันนี้เหลือ /200" และ "เดือนนี้เหลือ /1000"
- กรอกจำนวนเงิน → กด **คำนวณ** → แสดง "รัฐช่วย" และ "คุณจ่าย"
- กด **บันทึก** → หักยอดสิทธิ์ และเพิ่มลงประวัติการใช้จ่าย
- ประวัติการใช้จ่าย (วันที่:เวลา, ยอดซื้อ, รัฐช่วย, คุณจ่าย) ลบรายรายการ หรือ ล้างทั้งหมดได้
- จดจำข้อมูล: ปิดเว็บแล้วกลับมาเปิดใหม่ ค่าคงเหลือและประวัติยังอยู่ครบ

## วิธีคำนวณ
ให้ `P` = จำนวนเงินที่ซื้อ, `dailyLeft`/`monthlyLeft` = สิทธิ์ที่เหลือ
- `รัฐช่วย = min(0.6 × P, dailyLeft, monthlyLeft)` (จำกัดด้วยเพดานที่เหลือ)
- `คุณจ่าย = P − รัฐช่วย`

เมื่อบันทึก ระบบจะหัก `รัฐช่วย` ออกจากทั้งสิทธิ์รายวันและรายเดือน
สิทธิ์รายวันรีเซ็ตเป็น 200 เมื่อข้ามวัน และรายเดือนรีเซ็ตเป็น 1,000 เมื่อข้ามเดือน (ไม่ทบยอดข้ามเดือน)

## การจดจำข้อมูล (ไม่ต้อง login)
ใช้ **`localStorage`** ของเบราว์เซอร์ ซึ่งเป็นแนวทางยอดนิยมสำหรับเว็บ static ที่ไม่มี backend:
- ข้อมูลถูกเก็บในเครื่อง/เบราว์เซอร์ของผู้ใช้ จึงจำได้แม้ปิดเว็บไปแล้ว
- ข้อจำกัด: ผูกกับเบราว์เซอร์/เครื่องนั้น ๆ และจะหายหากผู้ใช้ล้างข้อมูลเบราว์เซอร์
- หากต้องการ sync ข้ามเครื่องในอนาคต สามารถต่อยอดด้วย Backend-as-a-Service เช่น Firebase / Supabase (พร้อม anonymous ID)

ข้อมูลถูกเก็บใต้คีย์ `thpp_state_v1` เป็น JSON

## รันบนเครื่อง (Local)
เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ตรง ๆ หรือเปิดเซิร์ฟเวอร์ง่าย ๆ:
```bash
python -m http.server 8000
# แล้วเปิด http://localhost:8000
```

## Deploy ขึ้น GitHub Pages
1. สร้าง repository ชื่อ `ThaiHelpsThaiPlus` บน GitHub แล้ว push โค้ดขึ้น branch `main`
   ```bash
   git add .
   git commit -m "ไทยช่วยไทย พลัส landing page"
   git push origin main
   ```
2. ไปที่ **Settings → Pages**
3. ที่ **Source** เลือก **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → Save
4. รอสักครู่ จะได้ URL: `https://<username>.github.io/ThaiHelpsThaiPlus/`

> หมายเหตุ: path ของไฟล์ใช้แบบ relative (`assets/...`, `css/...`, `js/...`) จึงทำงานได้ถูกต้องใต้ subpath ของ GitHub Pages
